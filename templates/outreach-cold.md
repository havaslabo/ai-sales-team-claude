# コールドアウトリーチシーケンス — 3通のメールフレームワーク（日本BtoB向け）

> 紹介や共通の接点がない状態からのコールドアウトリーチです。日本のBtoB商談では
> コールドメールは最終手段であり、まず紹介ルートを探すことを推奨します。
> このシーケンスは、紹介が得られなかった場合にのみ使用してください。
> すべての [PLACEHOLDER] フィールドを見込み客に合わせた情報に置き換えてください。

---

## コールドアウトリーチ実施前の確認

以下をすべて確認してから、このテンプレートを使用してください：

- [ ] 社内・既存顧客に紹介できる人物がいないことを確認した
- [ ] 共通の業界団体・商工会議所のつながりがないことを確認した
- [ ] 展示会・セミナーでの対面接点がないことを確認した
- [ ] それでも接触が必要なため、コールドアウトリーチを選択する

---

## タイムライン概要

| 日程 | チャネル | アクション |
| ---- | --------- | ------------------------------ |
| 1    | メール    | メール1：初回接触              |
| 3    | メール    | メール2：価値提供              |
| 5    | 電話      | テレアポフォローアップ         |
| 7    | メール    | メール3：クロージング          |
| 10   | 電話      | 最終フォローアップ電話（任意） |

## メール1：初回接触（1日目）

**目的：** 礼儀正しく接点を作る。まず信頼できる存在として認識してもらう。売り込みは禁物。

**件名の形式：**
- `株式会社[COMPANY_NAME] [LAST_NAME]様へ / [用件の概要]`
- `[COMPANY_NAME]様 / [TOPIC]についてのご連絡`

**本文：**

```
株式会社[COMPANY_NAME]
[DEPARTMENT]
[LAST_NAME] [FIRST_NAME] 様

突然のご連絡、失礼いたします。
[YOUR_COMPANY] の [YOUR_NAME] と申します。

[PERSONALIZED_OBSERVATION — 先方の取り組み・ニュース・登壇への具体的な言及。
例：「先日、[業界誌・イベント名] にて貴社の [SPECIFIC_INITIATIVE] について
拝見し、大変興味深く読ませていただきました。」1〜2文。]

弊社は [TARGET_PERSONA] 様の [RELEVANT_CHALLENGE] に関して、
[HOW_YOU_HELP] のお手伝いをしております。
[ONE_RELEVANT_CASE — 類似業界での課題解決の経験を押しつけにならない形で。
例：「[業界]の企業様と [SPECIFIC_APPROACH] に取り組んできた経験があります。」]

もしご興味をお持ちいただけるようであれば、一度 30分ほど情報交換の場を
いただけますでしょうか。
ご多用の折、誠に恐れ入りますが、ご検討いただければ幸いです。

よろしくお願いいたします。

[YOUR_NAME]
[YOUR_TITLE]
[YOUR_COMPANY]
[YOUR_EMAIL]
[YOUR_PHONE]
```

**CTA：** 情報交換の場を丁寧にお願いする（デモや提案の要求はしない）
**送信タイミング：** 火曜〜木曜、午前9〜11時（月初・月末・年末年始・お盆は避ける）

## メール2：価値提供（3日目）

**目的：** 何も求めずに純粋な価値を提供する。有益な情報・事例を届け、信頼を築く。返信を求めない。

**件名の形式：**
- `Re: [PREVIOUS_SUBJECT]`（スレッドを継続）
- `[RESOURCE_TITLE] — ご参考になれば / [YOUR_COMPANY] [YOUR_NAME]`

**本文：**

```
[LAST_NAME] 様

先日ご連絡いたしました [YOUR_COMPANY] の [YOUR_NAME] です。
ご返信のご負担をおかけするつもりはございませんが、
[COMPANY_NAME] 様のお役に立てそうな情報がありましたので
お送りさせていただきます。

[DESCRIBE_RESOURCE — 先方の業界・役割にとって本当に有益なレポート・
業界動向・事例・フレームワークを説明してください。
押しつけにならず、純粋に役立つ情報として。2〜3文。
例：「[業界] 向けに [TOPIC] に関する調査レポートが公開されており、
特に [SPECIFIC_FINDING] の点が [COMPANY_NAME] 様の取り組みにも
関連するかと思い、ご共有いたします。」]

[LINK_TO_RESOURCE]

ご参考になれば幸いです。

[YOUR_NAME]
[YOUR_COMPANY]
```

**CTA：** なし — 純粋な価値提供。返信を求めない。
**送信タイミング：** 火曜〜木曜の午前中

---

## メール3：クロージング（7日目）

**目的：** 丁重に締めくくりつつ、将来の接触を可能にする。日本では3通以上は「しつこい」と受け取られるため、ここで完結させる。

**件名の形式：**
- `Re: [PREVIOUS_SUBJECT]`
- `ご連絡の件 — [YOUR_COMPANY] [YOUR_NAME]`

**本文：**

```
[LAST_NAME] 様

[YOUR_COMPANY] の [YOUR_NAME] です。
先日来よりご連絡を差し上げておりましたが、
本メールを最後のご連絡とさせていただきます。

[PAIN_POINT_OR_INITIATIVE] が今後の課題として浮上した際には、
ぜひお声がけいただけますと幸いです。
[YOUR_EMAIL] または [YOUR_PHONE] にてお気軽にご連絡ください。

[COMPANY_NAME] 様のご発展をお祈り申し上げます。

以上

[YOUR_NAME]
[YOUR_TITLE]
[YOUR_COMPANY]
[YOUR_EMAIL]
[YOUR_PHONE]
```

**CTA：** 将来の再接触の可能性を残す
**送信タイミング：** 火曜または水曜の午前中

## 電話フォローアップ（5日目・任意）

**切り出しトーク：**
```
「お忙しいところ失礼いたします。
[YOUR_COMPANY] の [YOUR_NAME] と申します。
先日、[用件の概要] につきましてメールをお送りしておりましたが、
ご確認いただけましたでしょうか。

もし差し支えなければ、[SPECIFIC_TOPIC] について
5分ほどお話しいただけますでしょうか。」
```

**電話のルール：**
- 午前10時〜12時、午後2時〜4時が比較的つながりやすい
- 担当者不在の場合は折り返しをお願いするか、かけ直す時間を確認する
- 1回の電話は5分以内を目安にする
- 強引に長く引き留めない

---

## シーケンスの注意事項

- **コールドは最終手段：** 紹介ルートが見つかった場合は outreach-warm.md に切り替える
- **件名は必ず宛名付きで：** 「株式会社〇〇 山田様へ」の形式を基本とする
- **価格・金額は出さない：** 最初の接触では月額や費用の話は一切しない
- **配信停止：** 「結構です」「不要です」などの返信は即座に対応し、すべてのシーケンスから除外する
- **CRM 記録：** すべてのタッチ・開封・返信を CRM に記録する
- **一時停止のトリガー：** 見込み客から返信があった場合（肯定・否定を問わず）はシーケンスを即停止する
- **メール間隔：** 最低48〜72時間空けること；日本では間隔が短すぎると逆効果
- **業界データソース：** リサーチには企業サイト、Wantedly、業界団体サイト、業界誌、INITIAL（entrepedia）、SPEEDA を活用する
- **社内CC：** 重要な初回コールドメールは、自社の上司または営業管理者にCCすること。日本企業では営業活動の上司への報告・共有が標準的
- **人事異動に注意：** 日本では4月に大規模な人事異動がある。担当者が変わっている可能性があるため、4月前後は担当者の確認が必要
