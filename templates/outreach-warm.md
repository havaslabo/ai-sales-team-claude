# ウォームアウトリーチシーケンス — 最優先戦略（日本BtoB向け）

> 日本のBtoB営業において、**紹介・ウォームイントロは最も効果的なアプローチ**です。
> 共通の知人・取引先からの紹介、同じイベント・業界団体でのつながりなど、
> 何らかの接点がある見込み客へのアウトリーチはコールドの数倍の成功率があります。
> コールドアウトリーチに進む前に、必ずこの戦略を最初に試みてください。
> すべての [PLACEHOLDER] フィールドを見込み客に合わせた情報に置き換えてください。

---

## ウォームな接点の種類と優先順位

| 接点の種類 | 優先度 | アプローチ方法 |
|----------|------|------------|
| 既存顧客・パートナーからの直接紹介 | 最高 | 紹介者にまず連絡し、紹介メールを依頼する |
| 共通の知人（業界人脈）からの紹介 | 高 | 紹介者経由で接点を作る |
| 同じ業界団体・商工会議所のメンバー | 高 | 共通の所属を明記してメール |
| 展示会・セミナーでの名刺交換後 | 高 | 「先日お会いした件で」と切り出す |
| 先方のコンテンツ（記事・登壇）への反応 | 中 | 具体的な内容への言及から始める |

## 紹介者を活用したアプローチ（推奨）

### 紹介依頼メール（紹介者へ送るメール）

紹介者に送るメール文例：

```
[REFERRER_LAST_NAME] 様

お世話になっております。[YOUR_NAME] です。

突然のお願いで恐縮ですが、[TARGET_COMPANY] の [TARGET_NAME] 様を
ご存じとのことで、ご紹介いただけないかと思いご連絡いたしました。

[REASON_FOR_INTRODUCTION — なぜ紹介してほしいか、1〜2文で具体的に]

もしお差し支えなければ、[TARGET_NAME] 様に弊社をご紹介いただける
メールをお送りいただくか、または三者間でメールでご紹介いただけますと
大変ありがたく存じます。

ご多用の折、誠に恐れ入りますがご検討いただければ幸いです。

よろしくお願いいたします。
[YOUR_NAME]
```

### 紹介者から見込み客への紹介メール例（三者間）

紹介者が送る形式：

```
[TARGET_LAST_NAME] 様

お世話になっております。[REFERRER_NAME] です。

突然のご連絡で恐縮ですが、信頼できる知人をご紹介させてください。
[YOUR_NAME] さんは [YOUR_COMPANY] の [YOUR_TITLE] で、
[BRIEF_DESCRIPTION_OF_YOUR_WORK] をされています。

[REASON_WHY_RELEVANT — なぜ紹介するか、1〜2文]

ぜひお二人でお話しいただければと思い、ご紹介させていただきました。
[YOUR_NAME] さん、[TARGET_LAST_NAME] 様にご連絡いただけますか？

よろしくお願いいたします。
[REFERRER_NAME]
```

## メール1：ウォームな接点の活用（1日目）

**目的：** 共有の接点を明確に確立し、信頼と親近感を自然に作る。

**件名の形式：**
- `株式会社[COMPANY_NAME] [LAST_NAME]様へ / [MUTUAL_CONNECTION]様よりご紹介いただきました`
- `[LAST_NAME]様 / [EVENT/COMMUNITY] でのご縁について — [YOUR_NAME]`
- `[EVENT名] にてご名刺をいただきました件 — [YOUR_COMPANY] [YOUR_NAME]`

**本文（紹介者あり）：**

```
株式会社[COMPANY_NAME]
[DEPARTMENT]
[LAST_NAME] [FIRST_NAME] 様

[MUTUAL_CONNECTION] 様よりご紹介いただき、ご連絡させていただきます。
[YOUR_COMPANY] の [YOUR_NAME] と申します。

[WARM_CONTEXT — 紹介者から伺った内容、共通の接点を自然に記載。
例：「[MUTUAL_CONNECTION] 様より、[COMPANY_NAME] 様が
[DOING_SOMETHING_RELEVANT] に取り組んでいらっしゃると伺いました。」
1〜2文。]

弊社は [RELEVANT_CHALLENGE] に関して、[TARGET_PERSONA] 様の
お力になっており、[COMPANY_NAME] 様のご状況にも
ご参考にしていただけるかと思いご連絡いたしました。

[ONE_RELEVANT_CASE — 類似業界での取り組み事例を簡潔に。1文。
「最近では [SIMILAR_COMPANY 様と] [SPECIFIC_APPROACH] に取り組んでまいりました。」]

ご都合のよろしい際に、まず 30分ほど情報交換の場をいただけますでしょうか。
ご多用の折、誠に恐れ入りますが、ご検討いただければ幸いです。

よろしくお願いいたします。

[YOUR_NAME]
[YOUR_TITLE]
[YOUR_COMPANY]
[YOUR_EMAIL]
[YOUR_PHONE]
```

**本文（イベント・展示会での名刺交換後）：**

```
株式会社[COMPANY_NAME]
[LAST_NAME] [FIRST_NAME] 様

先日は [EVENT_NAME] にてご名刺をいただき、ありがとうございました。
[YOUR_COMPANY] の [YOUR_NAME] です。

[SPECIFIC_MEMORY — 当日お話しした内容への具体的な言及。
例：「[TOPIC] についてご意見を伺えたこと、大変参考になりました。」
1〜2文。]

改めてご連絡いたしましたのは、弊社が [RELEVANT_CHALLENGE] に関して
[HOW_YOU_HELP] のお手伝いをしており、[COMPANY_NAME] 様のご取り組みとも
ご縁があるかと感じたためです。

よろしければ、改めて詳しくお話しする機会をいただけますでしょうか。

よろしくお願いいたします。

[YOUR_NAME]
[YOUR_TITLE]
[YOUR_COMPANY]
[YOUR_EMAIL]
[YOUR_PHONE]
```

**CTA：** 情報交換・意見交換をテーマにしたカジュアルなお願い（売り込みではなく）
**送信タイミング：** 火曜〜木曜、午前9〜11時

## メール2：価値提供（4日目）

**目的：** 先方の課題に対して高い関連性を持つ情報・事例を届け、専門性を示す。返信は求めない。

**件名の形式：**
- `Re: [PREVIOUS_SUBJECT]`（スレッドを継続）
- `[THEIR_SPECIFIC_CHALLENGE] に関する情報 — [YOUR_COMPANY]`
- `[RESOURCE_TITLE] — [COMPANY_NAME] 様のご参考に`

**本文：**

```
[LAST_NAME] 様

[YOUR_NAME] です。先日ご連絡いたしました件に関連して、
[COMPANY_NAME] 様のお役に立てそうな情報がありましたのでお送りします。

[DESCRIBE_VALUE — 本当に有益なリソース・業界データ・事例・フレームワークを
ご提供ください。以下のようなものが効果的です：
- [業界] に関連する調査レポートやデータポイント
- 類似企業での業務改善事例（金額よりも課題解決の文脈で）
- 先方の取り組みに参考になりそうなフレームワークや考え方
2〜3文。「弊社サービスのご案内」にならないよう注意する。]

[KEY_TAKEAWAY — 先方の状況に最も関連するポイントを1つ抜粋。
1〜2文。「[業界] 分野で [APPROACH] を実践されている企業では
[OUTCOME] のケースが増えています — [COMPANY_NAME] 様の
ご取り組みにも参考になるかと存じます。」]

ご参考になれば幸いです。もし詳しくお話しできればと思っておりますが、
お気持ちに応じてお返事いただければ十分です。

よろしくお願いいたします。

[YOUR_NAME]
[YOUR_COMPANY]
```

**CTA：** 詳細説明のソフトなオファー — 強引なミーティングの要求はしない
**送信タイミング：** 火曜〜木曜の午前中

## 電話フォローアップ（7日目）

**目的：** メールだけでなく、電話でも接点を作ることで印象を残す。

**切り出しトーク：**
```
「お忙しいところ失礼いたします。
[MUTUAL_CONNECTION] 様よりご紹介いただきました、
[YOUR_COMPANY] の [YOUR_NAME] と申します。

先日メールをお送りしておりましたが、ご確認いただけましたでしょうか。

もし差し支えなければ、[SPECIFIC_TOPIC] について
5〜10分ほどお話しいただけますでしょうか。」
```

**紹介なしの場合（展示会等）：**
```
「お忙しいところ失礼いたします。
先日 [EVENT_NAME] でご名刺をいただきました、
[YOUR_COMPANY] の [YOUR_NAME] です。

先日メールでご連絡させていただいておりましたが
ご確認いただけましたでしょうか。

よろしければ、改めてお話しする時間をいただけますでしょうか。」
```

## メール3：アポイントのお願い（10日目）

**目的：** 明確な価値提案とともに、具体的なミーティングを丁寧にお願いする。

**件名の形式：**
- `Re: [PREVIOUS_SUBJECT]`（スレッドを継続）
- `[SPECIFIC_TOPIC] についてのご相談 — [YOUR_COMPANY] [YOUR_NAME]`

**本文：**

```
[LAST_NAME] 様

[YOUR_NAME] です。これまで何度かご連絡を差し上げておりましたが、
最後のご連絡とさせていただきます。

[SPECIFIC_TOPIC — 先方の課題・状況に合わせた具体的なテーマ] について、
一度 30分ほどお話しする機会をいただければ幸いです。

[SPECIFIC_DELIVERABLE — 何を話したいか、先方にとって何が得られるか。
例：「[SIMILAR_COMPANY] 様での [SPECIFIC_APPROACH] の事例をご共有しながら、
[COMPANY_NAME] 様の [RELEVANT_CHALLENGE] へのアプローチについても
ぜひお聞かせいただければと思っています。」]

ご都合のよろしい日程をお知らせいただけますでしょうか：
- [OPTION_1]
- [OPTION_2]
- [OPTION_3]

上記以外のご都合のよい日程がございましたら、遠慮なくお申しつけください。

今のタイミングが難しければ、[TRIGGER_EVENT — 例：「年度計画のご検討時期」や
「新体制が落ち着いた後」] にあらためてご連絡いたしますのでご遠慮なくお知らせください。

よろしくお願いいたします。

[YOUR_NAME]
[YOUR_TITLE]
[YOUR_COMPANY]
[YOUR_EMAIL]
[YOUR_PHONE]
```

**CTA：** 複数の候補日を提示した丁寧なミーティングのお願い
**送信タイミング：** 火曜または水曜の午前9〜11時

## シーケンスの注意事項

- **紹介者へのフォロー（礼儀として必須）：** 連絡を取ったことを [MUTUAL_CONNECTION] 様に必ず報告する。結果（返信があったか等）も簡潔にお伝えする。
- **ウォームな接点が重要：** すべてのメールで共通の接点を自然に記載すること。接点を強調しすぎると不自然になるため注意。
- **トーン：** コールドアウトリーチより会話的・親密に。ただし初回から馴れ馴れしくなりすぎない。
- **価格・金額は出さない：** 月額や費用の話は最初の接触では不要。まず関係構築を優先する。
- **返信があった場合：** シーケンスを中止し、自然な会話に移行すること。1営業日以内に返信する。
- **メール3でも返信がない場合：** 30日待ってから、状況変化（トリガーイベント）があれば再度ご連絡する。
- **CRM 記録：** ウォームな接点のソース（誰からの紹介か、どのイベントか）を将来の参照のために必ず記録する。
- **業界データソース：** リサーチには企業サイト、Wantedly、業界団体サイト、業界誌、INITIAL（entrepedia）、SPEEDA を活用する。
