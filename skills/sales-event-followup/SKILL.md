# 展示会・イベント後フォロースキル

展示会・セミナー・業界イベントで名刺交換した後のフォローアップメールシーケンスを生成します。「先日はありがとうございました」から始まる自然な関係構築シーケンスで、日本BtoB営業における最も成功率が高い接点を最大化します。

日本のイベントフォローにおける重要な前提：
- 御礼メールは **翌営業日中** に送ることが日本のビジネスマナーです。遅くとも3営業日以内。
- 名刺交換直後のメールは「売り込み」ではなく「御礼・ご挨拶」が主目的です。
- フォローアップは段階的に。初回で商談を求めるのは逆効果になりやすい。
- 会話内容（ブース・セミナー・懇親会で話したこと）を必ず言及して「覚えています」を示す。

## 呼び出し方法

```
/sales event-followup <event_name> <company_or_contact>
```

`<event_name>` はイベント名（例：「Japan IT Week 2025」「〇〇業界セミナー」）、`<company_or_contact>` は会社名または担当者情報です。

---

## ステップ1：インプットの収集

以下の情報を収集します。情報が多いほどパーソナライズ度の高いメールが生成されます。

### 必須インプット

1. **イベント名**：正式名称（例：「Japan IT Week 2025春 ソフトウェア＆アプリ開発展」）
2. **イベント開催日**：名刺交換した日付
3. **相手の会社名・氏名・役職**：名刺情報
4. **自社名・送信者氏名・役職・連絡先**：署名情報

### オプションインプット（品質向上に大きく貢献）

5. **会話内容のメモ**：ブース・セミナー・懇親会で話した内容（課題・関心事・質問等）
6. **相手の関心領域**：興味を示した製品・機能・事例
7. **次のアクションの合意**：「資料を送ります」「改めてご連絡します」等の約束
8. **相手の業界・事業内容**：業種・会社規模・事業概要
9. **イベントの文脈**：出展者側か来場者側か、セミナー登壇者か聴衆か

### イベント種別の確認

以下のいずれかを確認し、適切なトーンと文面で生成します。

| 種別 | 例 | 特徴 |
|------|-----|------|
| 展示会・見本市 | Japan IT Week、Interop、CEATEC、業界展 | フォーマル、多くの名刺交換が前提 |
| 主催セミナー | 自社開催ウェビナー、勉強会 | 登壇内容への言及が有効 |
| 業界団体セミナー | 協会・団体主催のセミナー | 業界テーマへの共感を示す |
| 交流会・懇親会 | 業界懇親会、異業種交流会 | カジュアルなトーンが適切 |
| オンラインウェビナー | Zoom、Teams等でのオンライン開催 | 「画面越しにお会いしました」表現を活用 |

---

## ステップ2：フォローアップシーケンスの生成

3段階のメールシーケンスを生成します。各メールの目的・トーン・タイミングを厳守します。

### メール1：御礼メール（翌営業日・最優先）

**目的：** 良い第一印象を定着させる。「この会社は丁寧だ」と思ってもらう。

**送信タイミング：** 名刺交換翌営業日（遅くとも3営業日以内）

**件名フォーマット：**
```
[イベント名] にてお名刺をいただきました件 / [自社名] [氏名]
```

**本文の構成：**

1. **書き出し（必須）：**「先日は[イベント名]にてお名刺をいただき、誠にありがとうございました。」
2. **会話の言及（会話メモがある場合）：**「〇〇についてお話しくださった件、大変参考になりました。」または「〇〇のご質問をいただき、改めて考えるきっかけをいただきました。」
3. **自社・製品紹介（1〜2行のみ）：** 売り込まない。簡潔に何をしている会社かのみ。
4. **次のアクション（柔らかいCTA）：**「改めてご挨拶の機会をいただけますでしょうか。」または「ご都合のよろしいタイミングで、少しお時間をいただけますと幸いです。」
5. **署名：** 会社名・氏名・役職・電話番号・メールアドレス・会社URL

**トーン：** 丁寧・誠実。売り込みゼロ。感謝を主体にする。

---

### メール2：フォローアップメール（1〜2週間後）

**目的：** 有益な情報提供で「この人は役に立つ」という印象を作る。接点を維持する。

**送信タイミング：** 御礼メール送信から1〜2週間後

**件名フォーマット：**
```
[相手の関心事に関連する内容] のご参考情報 / [自社名] [氏名]
```
または
```
先日のご連絡に関して / [自社名] [氏名]
```

**本文の構成：**

1. **書き出し：**「先日はご返信をいただき、ありがとうございました。」または「先日の[イベント名]以来、いかがお過ごしでしょうか。」
2. **情報提供（メイン）：** 相手の業界・関心事に関連する有益な情報を1つ。事例・レポート・業界ニュース・ホワイトペーパー等。「ご参考になれば幸いです」スタイルで提供する。売り込まない。
3. **軽いCTA（任意）：**「ご感想などお聞かせいただけますと幸いです。」または次の接点を提案（ウェビナー案内等）。

**トーン：** 「お役に立ちたい」姿勢。プレッシャーを与えない。

---

### メール3：提案メール（3〜4週間後）

**目的：** 課題感が確認できた場合に商談機会を設ける。

**送信タイミング：** フォローアップメールから2〜3週間後（課題感が確認できた場合のみ送信）

**送信条件：** 以下のいずれかが確認できた場合のみ送信する。
- メール2への返信で課題・関心を言及された
- 資料ダウンロード・ウェビナー参加等のエンゲージメントがあった
- 会話メモに「現在〇〇を検討している」「〇〇が課題」とあった

**件名フォーマット：**
```
[課題・テーマ] に関するご相談 / [自社名] [氏名]
```

**本文の構成：**

1. **書き出し：**「〇〇について関心をお持ちとのこと、改めてお役に立てればと思いご連絡いたしました。」
2. **課題への共感：** 相手が言及した課題・関心事を具体的に引用する
3. **提案（1〜2行）：** 解決できることを簡潔に提示
4. **CTA（具体的に）：**「30分ほどお時間をいただけますでしょうか。オンライン・対面どちらでも対応可能です。」
5. **日程調整の選択肢（任意）：** 「来週〇日〇時または〇日〇時はいかがでしょうか。」

**トーン：** 自信を持ちつつ押しつけがましくない。相手の都合を最優先にする。

---

## ステップ3：ファイルの保存

生成したフォローアップシーケンスを以下のファイル名で保存します。

```
EVENT-FOLLOWUP-[会社名].md
```

例：`EVENT-FOLLOWUP-株式会社山田製作所.md`

保存後、以下を担当者に案内します。
- 各メールの送信タイミング（翌営業日・1〜2週間後・3〜4週間後）
- カスタマイズが必要な箇所（[イベント名]・[氏名]等のプレースホルダー）
- 送信前確認ポイント（宛先・件名・署名の確認）
- メール3の送信条件（課題感の確認）

---

## 重要なルール

- **御礼メールは翌営業日中に送る。** タイミングが遅れると印象が薄れる。
- **初回は売り込まない。** 御礼と自己紹介のみ。製品説明は1〜2行に留める。
- **会話内容を必ず言及する。** 「覚えています」を示すことで差別化できる。
- **段階的に関係を構築する。** 3通のシーケンスで信頼を積み上げてから商談へ。
- **相手の業種・役職・会話内容に合わせてパーソナライズする。** 汎用的な文面は開封されない。
- **日本語・敬体（です・ます調）で統一する。** 相手の役職・業界に合わせたフォーマル度を調整する。
- **EVENT-FOLLOWUP-[会社名].md に保存する。** ファイル名は固定フォーマットに従う。
