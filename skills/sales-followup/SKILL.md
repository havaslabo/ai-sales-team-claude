# フォローアップシーケンスジェネレーター

初回コンタクト後の見込み客に対して、戦略的なフォローアップメールシーケンスを生成します。これはコールドアウトリーチではありません。ミーティング、デモ、提案、または過去の会話の後のフォローアップです。すべてのフォローアップは新しい価値を加え、会話の具体的なポイントを参照し、明確な次のステップを含む必要があります。

日本のBtoB営業における重要な前提：
- フォローアップの頻度は2〜3週間に1回が礼儀として適切です。毎週のフォローアップは押しつけがましく関係を損なう可能性があります。
- 「返信なし = 断り」は日本では必ずしも当てはまりません。多忙で見ていないケース・上司への相談中・社内稟議中であることが多いです。粘り強く、丁寧に、価値を提供し続けることが重要です。
- メール文体は丁寧な敬語を使用し、相手の立場を尊重した表現にしてください。

## 呼び出し方法

```
/sales followup <prospect>
```

`<prospect>` には会社名、担当者名、URL、または見込み客とフォローアップのコンテキストの説明を指定します。

## ステップ1: フォローアップのコンテキストを収集する

シーケンスを生成する前に、以下の情報を収集します。提供されていない場合はユーザーに確認します:

1. 見込み客の**氏名**と**会社名**
2. **前回のインタラクションの種類**: ミーティング、デモ、提案、音信不通（返信なし）、長期育成（Long-Term Nurture）
3. **前回のインタラクションの日付**
4. 前回のインタラクションからの**主要な議論ポイント**（最低3つの具体的な内容）
5. 見込み客が**述べた課題**または目標
6. **合意された次のステップ**（ある場合）
7. 見込み客の**役職とシニアレベル**（トーンとコンテンツに影響）
8. **案件ステージ**: 初期探索、積極的な評価、意思決定間近、停滞中
9. **見込み客の温度感**: ホット（積極的にエンゲージ中）、ウォーム（興味はあるが進みが遅い）、クール（関与が低い）、コールド（返信なし）
10. **製品・サービス**と議論した具体的なソリューション

作業ディレクトリに以前の分析ファイルが存在する場合（PROSPECT-ANALYSIS.md、COMPANY-RESEARCH.md、LEAD-QUALIFICATION.md、DECISION-MAKERS.md、OUTREACH-SEQUENCE.md）、それらを読み込み、関連する知見をフォローアップのコンテキストに自動的に組み込みます。

## ステップ2: フォローアップシナリオを選択する

前回のインタラクションの種類に基づいて、以下から該当するシナリオを選択します。ユーザーが指定しない場合は、どのシナリオが最も適切かを確認します。

---

### シナリオ1: ミーティング後フォローアップ（3通）

使用場面: ディスカバリーコール、初回ミーティング、またはストラテジーセッションが行われた後。

**メール1 — サマリー＋次のステップ**（送信: ミーティング当日、終了から2〜3時間以内）
- 件名フォーマット: 「本日のご面談のお礼と議事概要」など、ミーティングの内容を示す件名
- 冒頭: 「本日はお忙しい中、貴重なお時間をいただきありがとうございました。」など感謝の言葉から始める
- ミーティングで印象に残った1つの具体的な洞察や課題から始める（しっかり傾聴していたことを示す）
- 議論した3つの重要ポイントを箇条書きでまとめる（「議事概要」の体裁）
- 合意された次のステップを具体的な日程とともに再確認する
- ミーティング中に約束した資料を添付またはリンクする
- 締め: 「ご不明な点がございましたら、お気軽にお問い合わせください。引き続きどうぞよろしくお願いいたします。」
- 長さ: 100〜150字（日本語）

**メール2 — 価値の強化**（送信: メール1の2〜3週間後）
- 件名フォーマット: 「【ご参考資料のご送付】〇〇に関する情報」など、内容がわかる件名
- 冒頭: 「先日はご面談いただき、誠にありがとうございました。その後、いかがでしょうか。」
- 相手が述べた課題の1つに直接対処する関連資料（事例・データ・業界情報）をご提供する
- その資料が相手の具体的な課題解決にどのように役立つかを簡潔に説明する
- ミーティングで議論されなかった1つの新しい視点や情報を含める
- 次のステップへの丁寧な打診: 「ご参考になれば幸いです。もしよろしければ、改めてご状況をお聞かせいただけますでしょうか。」
- 長さ: 100〜150字（日本語）

**メール3 — 意思決定の後押し**（送信: メール2の2〜3週間後）
- 件名フォーマット: 「〇〇についてのご検討状況のご確認」
- 冒頭: 「先日ご連絡差し上げました〇〇社の△△でございます。ご検討の状況はいかがでしょうか。」
- 相手の状況に関連する数値や事例を共有する（同業他社の活用効果など）
- 相手のスケジュールや検討状況を丁寧に確認する
- 具体的な日程を提案しつつ、相手のペースを尊重する表現を使う: 「ご都合よろしければ、〇月〇日ごろに改めてご説明の機会をいただけますでしょうか。ご都合の良い日時をお知らせいただければ調整いたします。」
- 長さ: 80〜120字（日本語）

### シナリオ2: デモ後フォローアップ（4通）

使用場面: 製品デモ、ウォークスルー、またはテクニカルプレゼンテーションが実施された後。

**メール1 — 振り返り＋リソース**（送信: デモ当日、終了から2〜3時間以内）
- 件名フォーマット: 「本日のデモンストレーションのお礼と資料のご送付」
- 冒頭: 「本日はお時間をいただき、誠にありがとうございました。」
- デモ中に最も関心を示した、または最も質問が多かった具体的な機能・画面を参照して始める
- 相手のニーズに対応する実演した3〜4つの機能・ポイントを箇条書きでまとめる
- 添付: デモ録画（利用可能な場合）、関連資料、セットアップガイド、または概要資料
- デモ中に約束したフォローアップの回答を含める
- 締め: 「ご不明な点やご質問がございましたら、いつでもお気軽にお申し付けください。」
- 長さ: 100〜150字（日本語）

**メール2 — ご懸念への対応**（送信: メール1の2〜3週間後）
- 件名フォーマット: 「先日のデモに関するご質問への補足のご案内」
- 冒頭: 「先日はデモンストレーションにお時間をいただきありがとうございました。その後いかがでしょうか。」
- デモ中に提起された上位1〜2つのご懸念を丁寧に取り上げ、補足情報をご提供する
- 「他社様でも同様のご懸念をいただくことがございます。弊社の〇〇という対応実績がございますのでご参考ください。」という形で解決事例を示す
- 具体的な証拠（数値・導入事例・推薦コメント）を含める
- 参考ユーザー企業のご紹介のオファーでクローズ: 「ご希望でしたら、類似業種でご活用いただいているお客様のご担当者をご紹介することも可能でございます。」
- 長さ: 100〜150字（日本語）

**メール3 — 導入事例のご紹介**（送信: メール2の2〜3週間後）
- 件名フォーマット: 「【事例のご紹介】〇〇業界における活用事例」
- 冒頭: 「先日はご連絡差し上げました△△でございます。」
- 相手と類似した企業（同業界・同規模・同課題）からの事例をご紹介する
- 構成: 直面した課題 → 弊社ソリューションの活用内容 → 具体的で測定可能な成果
- 「御社と同様のご状況からスタートされたお客様の事例でございます。ご参考になれば幸いです。」と関連付ける
- クロージング: 「御社においても同様のご支援ができればと考えております。引き続きご検討のほど、よろしくお願いいたします。」
- 長さ: 100〜130字（日本語）

**メール4 — ご検討状況のご確認**（送信: メール3の2〜3週間後）
- 件名フォーマット: 「ご検討状況のご確認」
- 冒頭: 「いつもお世話になっております。△△でございます。先日よりご連絡差し上げておりますが、ご検討状況はいかがでしょうか。」
- 押しつけがましくならない形で、意思決定のタイムラインと状況を丁寧に確認する: 「ご多用の折、大変恐縮ではございますが、検討状況を教えていただけますでしょうか。」
- 社内での検討支援を申し出る: 「社内でのご検討にお役立ていただける資料（比較表・費用対効果試算など）をご用意することも可能でございます。お気軽にご相談ください。」
- 誠実な表現で、相手のペースを尊重した形でクローズする
- 長さ: 80〜120字（日本語）

### シナリオ3: 提案後フォローアップ（5通）

使用場面: 正式な提案書、見積書、または価格資料が送付された後。

**メール1 — 提案書の送付**（送信: 提案書と同時に）
- 件名フォーマット: 「【ご提案書のご送付】〇〇に関するご提案」
- 冒頭: 「いつもお世話になっております。このたびは弊社サービスについてご検討いただき、誠にありがとうございます。」
- 短く丁寧に — 提案書の内容をご確認いただくようお願いする
- 最初にご確認いただきたい重要な2〜3セクションを案内する
- ご説明の場のご提案: 「ご不明点や詳細につきましては、改めてご説明の機会をいただければ幸いです。〇月〇日ごろいかがでしょうか。」
- 提案書の有効期間に言及（該当する場合）
- 長さ: 80〜120字（日本語）

**メール2 — ご説明のご提案**（送信: メール1の1〜2週間後）
- 件名フォーマット: 「先日のご提案書についてのご確認」
- 冒頭: 「先日はご提案書をお送りいたしました〇〇でございます。ご確認の状況はいかがでしょうか。」
- 押しつけがましくなく、ご確認いただけたかどうかを確認する
- 「ご不明点やご質問がございましたら、15〜30分ほどオンラインでご説明する機会をいただければ幸いです。ご都合の良い日時をお知らせいただければ幸いです。」
- よくいただくご質問（価格・スコープ・スケジュール等）への補足情報を先回りしてご提供する
- 長さ: 80〜120字（日本語）

**メール3 — 価値を加えるご案内**（送信: メール2の2週間後）
- 件名フォーマット: 相手の業界・課題に関連する情報提供の件名
- このメールでは提案書について一切言及しない
- 代わりに、相手のビジネス課題に関連する本当に価値ある情報・事例・業界データをご提供する
- 取引を成立させることだけでなく、相手の成功を考えていることを示す
- さりげない関連付け: 「この情報を拝見し、先日お話しいただいた〇〇の件を思い出しました。少しでもご参考になれば幸いです。」
- 長さ: 80〜120字（日本語）

**メール4 — ご検討状況の直接確認**（送信: メール3の2週間後）
- 件名フォーマット: 「〇〇についてのご検討状況のご確認」
- 冒頭: 「大変お世話になっております。△△でございます。ご多用の中、大変恐縮ではございますが、現在のご状況をお教えいただけますでしょうか。」
- 丁寧かつ率直に確認する: 「ご検討の進捗や、社内での調整状況などお聞かせいただければ幸いです。」
- 意思決定のご支援を申し出る: 「社内でのご説明にお役立ていただける追加資料のご用意も可能です。何かございましたら、お気軽にお申し付けください。」
- Yes/Noでお答えいただきやすい質問で締める
- 長さ: 80〜100字（日本語）

**メール5 — 丁寧な締めくくり**（送信: メール4の2〜3週間後）
- 件名フォーマット: 「引き続きのご検討のお願い」または「一度ご連絡させてください」
- これがシーケンスの最後のメールです。日本では「縁切りメール（ブレイクアップメール）」の概念はなじまないため、丁寧な締めくくりとして位置づける。
- タイミングが合わない可能性を丁寧に認める: 「ご多用の折、ご検討にお時間をいただいていることと存じます。」
- 扉を開けておく: 「今後、ご状況が変わりましたら、またいつでもお声がけいただければ幸いです。引き続きどうぞよろしくお願いいたします。」
- 日本の商慣習として、「もし現時点でご縁がなくても、将来また機会があれば」という姿勢で誠実に締めくくる
- 長さ: 60〜100字（日本語）

### シナリオ4: 返信なしリカバリー（3通）

使用場面: 以前のエンゲージメント後、見込み客から返信がない場合。日本では「返信なし = 断り」とは限らないため、丁寧かつ継続的にアプローチする。

**メール1 — 別角度からのアプローチ**（送信: 最後の未返信メールの2週間後）
- 件名フォーマット: 「【ご参考情報のご送付】〇〇に関する最新事例」など、価値ある情報を示す件名。「確認のご連絡」や「フォローアップ」は避ける。
- トーンを変えて、新しい価値ある情報を提供する形でアプローチする
- 過去のメールへの言及は最小限にし、「先日ご連絡差し上げました〇〇でございます」程度にとどめる
- 答えやすいシンプルな確認をする: 「お手すきの際に、ご関心がおありかどうかだけでもお聞かせいただければ幸いです。」
- 長さ: 60〜100字（日本語）

**メール2 — 新しい情報の提供**（送信: メール1の2〜3週間後）
- 件名フォーマット: 新しい具体的な情報を前に出す件名
- まったく新しい内容を提供する — 業界の新動向・新機能・新しい事例・季節の話題
- 「この情報をご覧になり、〇〇の件でお話しいただいたことを思い出しました。ご参考になれば幸いです。」
- 丁寧な再エンゲージ: 「ご関心がございましたら、改めてご連絡いただければ幸いです。ご都合が合いましたらいつでも対応いたします。」
- 長さ: 60〜80字（日本語）

**メール3 — 丁寧な締めくくり**（送信: メール2の3週間後）
- 件名フォーマット: 「今後ともどうぞよろしくお願いいたします」または「ご縁をいただければ幸いです」
- 丁寧かつ誠実に締めくくる: 「何度かご連絡差し上げましたが、ご多用の中ご返信をいただけておらず、タイミングが合わなかったのかもしれません。」
- 一旦のご連絡を控える旨をお伝えする: 「今後はご不便をおかけしないよう、定期的なご連絡は一旦控えさせていただきます。」
- 丁寧に扉を開けておく: 「ご状況が変わられた際や、お役に立てることがございましたら、いつでもお声がけください。今後ともどうぞよろしくお願いいたします。」
- 相手のビジネスや目標への具体的な参照とともに、誠実な好意で終わる
- 長さ: 60〜100字（日本語）

### シナリオ5: 育成シーケンス（継続的な月次送信）

使用場面: 見込み客は良い適合者だが、今すぐ購入する準備ができていない。うんざりさせずにトップオブマインドを維持する。日本では長期的な関係構築が重要で、2〜3ヶ月に1度のペースが礼儀的。

6通の月次メールを生成します。各メールは単独でも本当に役立つコンテンツである必要があります。購入しなくても受け取って良かったと思えるものでなければなりません。

**月次メールパターン**（送信: 1〜2ヶ月に1回）
- 毎月これらのコンテンツタイプを交互に:
  1. **業界インサイト**: 業界に関連するトレンド、データポイント、または分析を共有（業界紙・調査レポートの引用など）
  2. **リソースシェア**: 役立つ資料・ツール・テンプレート・ガイド（自社コンテンツ以外でも可）
  3. **導入事例のご紹介**: 同業種の企業からの簡潔な成功事例
  4. **情報提供**: 業界の課題や市場動向についての情報提供
  5. **イベント・セミナーへのご案内**: ウェビナー、展示会、業界セミナー、レポートへのご招待
  6. **個人的なご確認**: 相手のビジネスに関する具体的な内容を参照した、誠実で簡潔なご挨拶

- 各育成メールは以下を満たす必要があります:
  - 件名自体が価値を提供する（「ご確認のご連絡」や「ご報告」ではなく）
  - セールスピッチなしで価値を提供する
  - 1つのさりげない関連付けを含める（最大1文）
  - 低プレッシャーの締め: 「ご参考になれば幸いです」または「ご興味がございましたら」— 「お打ち合わせをお願いします」は避ける
  - 200字以内（日本語）

## ステップ3: マルチチャネル統合

選択したシーケンスの各メールに対して、以下の付随するタッチポイントも生成します:

### 電話フォローアップ（テレアポ）スクリプト

シーケンスの重要な場面（初回接触後・提案書送付後）に向けた30〜60秒の電話スクリプトを生成します。日本では電話でのフォローアップがメールよりも確認・関係構築に有効なケースがあります。

**電話スクリプトテンプレート:**
```
「お電話失礼いたします。〇〇株式会社の△△と申します。
先日〔先日お送りしたメール/先日のご面談〕に関しましてご連絡差し上げました。
〔1文 — 通話の理由、具体的な会話ポイントと結びついた内容〕。
〔1文 — 新しい情報やご支援できること〕。
詳細は改めてメールでもご案内いたします。
もしよろしければ〔具体的な日程の提案〕、お時間をいただけますでしょうか。
お手数ですが、折り返しご連絡いただければ幸いです。
〔自社名・氏名・連絡先〕でございます。よろしくお願いいたします。」
```

電話スクリプトのルール:
- 声に出して読んで30〜60秒以内
- 丁寧で自然なトーン
- 以前の会話からの1つの具体的な内容を参照する
- 詳細については付随するメールへ誘導する
- 無理に当日の返事を求めない — 「ご都合のよい時に」という姿勢で

### メール補足テンプレート

電話後に送るフォローアップメールのテンプレートも生成します。電話とメールを組み合わせたアプローチが日本のBtoB営業では効果的です。

## ステップ4: カデンスの推奨

案件ステージと見込み客の温度感に基づいて、日本のBtoB営業に適したカデンスを推奨します。

### カデンスマトリックス（日本BtoB向け）

## ステップ5: 丁寧な締めくくりのベストプラクティス

シーケンスの最終メールを生成する際は、日本のビジネス文化に合わせた以下のルールに従います。日本では「ブレイクアップメール（縁切りメール）」の文化はなじまないため、関係を大切にした丁寧な締めくくりを使用します。

1. **礼儀正しく誠実に**: 相手に責任を感じさせたり、プレッシャーをかけたりしない。
2. **状況を丁寧に認める**: 「何度かご連絡差し上げましたが、ご多用の折、大変恐縮でございます」という表現が自然。
3. **扉を大切に開けておく**: 「今後ともお声がけいただければ幸いです」「ご縁がございましたら、またいつでもお気軽にご連絡ください」という表現でつながりを維持する。
4. **日本的な「間」を大切に**: 定期的なご連絡を一旦控える旨を伝え、相手のペースを尊重する姿勢を示す。
5. **ポジティブな言葉で終わる**: 相手のビジネスの発展を願う一文を添える。
6. **シーケンスの中で最も短いメールにする**: 丁寧な締めくくりメールは200字程度。
7. **将来の関係を大切に**: この見込み客は6〜12ヶ月後に顧客になる可能性や、別の機会でつながる可能性があります。日本のBtoBはリピート・紹介が重要です。

## ルールと制約

1. **すべてのメールに新しい価値を加える必要があります。** 単純な「ご確認のご連絡」は送信しない。新しく共有できる情報がない場合は送信しない。
2. **具体的な会話ポイントを参照する。** 汎用的なフォローアップは削除されます。すべてのメールは前回のインタラクションからの具体的な何かを参照する必要があります。
3. **1通のメールにつき1つの明確な次のステップ。** 複数のご依頼をしない。1通のメール = 1つのご案内。
4. **丁寧な日本語敬語を使用する。** 相手の立場・役職に応じた適切な敬語レベルを使用する。
5. **適切な緊急感。** 緊急感は本物でなければなりません（年度末・予算申請時期・導入スケジュール）— 作られた緊急感や操作的な表現は使用しない。
6. **パーソナライゼーションは必須です。** 見込み客の氏名・会社名・具体的な課題・会話ポイントを使用する。最終出力に[PLACEHOLDER]の括弧は残さない。
7. **プロフェッショナルで誠実なトーン。** 信頼できるパートナーとして書く。過度に形式的すぎず、かつカジュアルすぎない、日本のビジネス文体を使用する。
8. **操作的な戦術を使わない。** 偽りの希少性・罪悪感を抱かせること・プレッシャーをかける表現は不可。
9. **見込み客の時間を尊重する。** 返信の強要・過剰なフォローアップは避け、価値提供を通じた長期的な関係構築を目指す。
10. **以前の分析ファイルが存在する場合**、そのデータを組み込む。PROSPECT-ANALYSIS.md、COMPANY-RESEARCH.md、または作業ディレクトリの他の出力ファイルで既に利用可能な情報をユーザーに繰り返させない。
